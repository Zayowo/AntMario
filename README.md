# Formica
Un plateformer 2D, reprenant les méchaniques de Mario dans un monde tout droit sorti de Berserk.
Un engine custom C++ avec la librarie SFML, basé sur un Entity Component System (Les components sont des systèmes en eux-mêmes) a été crée pour le projet.

## Commandes
- Droite : D
- Gauche : Q
- Saut : Espace
- Sprint : (Maintenir)Shift 

## But du jeu

Compléter le niveau et surmonter les différents obstacles sur votre chemin. Récoltez des pièces (si vous aimez à ce point l'argent), éliminer les ennemis _et récoltez leurs sang pour débloquer des nouveaux moyens de se déplacer (sous forme d'orbs)_.

Accrochez-vous et déplacez-vous au plafond, effectuez des double sauts pour gravir les falaises, (et plus à venir...)

Mais attention! Le sang récolté est consommé à chacune de ses nouvelles actions, utilise votre nouvelle énergie à bon escient.

## Bugs de l'alpha

Et oui, version alpha, incomplète et peut-être instable, mais bon, que voulez-vous... C'est jouable! :)

- Le niveau ne possède pas de fin (Pas de victoire/game over)
- Il n'y a pas _encore_ de power-ups dans le niveau (même si les états existent déjà...)
- Les "Goombas" ne font pas de dégâts et vous pousse
- Si vous tombez dans le vide, vous tombez à l'infini (Avec le Menu pause, vous pouvez retourner au menu et recommencer la partie!)
- Des soucis par-ci par-là, je vais pas tout lister!!
